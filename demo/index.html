What is happening
The system runs a live SHA-256 integrity check on every memory and knowledge file. Each file is rehashed and compared against the stored manifest. If a single byte has changed anywhere — the root hash changes and the status flags CHANGED.

This is cryptographic tamper detection, computed in real time.
Workspace
/secure/ai-memory/workspace/ is the persistent memory layer — the directory containing all identity files, the knowledge tree, and the integrity manifest. This is what PCIS protects. Everything the AI knows about your clients, your products, and your compliance history lives here — signed, dated, and tamper-evident.
